# Alternatives Overview — Activos Alternativos

Análisis de activos no tradicionales para inversores institucionales y family offices.

## Cuando usar
- "analiza REITs europeos"
- "inversión en infraestructuras"
- "cripto para institucionales"
- "alternativos para diversificar"
- "qué peso dar a alternativos en mi cartera"

## Clases de alternativos

### REITs (Real Estate Investment Trusts)
Tickers principales:
- Simon Property Group (retail US): SPG
- Prologis (logística US): PLD
- Vonovia (residencial Alemania): VNA.DE
- Merlin Properties (oficinas España): MRL.MC
- Unibail-Rodamco (comercial Europa): URW.AS

Métricas clave: FFO, P/FFO, NAV discount, ocupación, yield.

### Infraestructura
ETFs/empresas:
- iShares Global Infra: IGF
- Ferrovial: FER.MC
- ACS: ACS.MC
- Atlantia (Italia): ATL.MI

### Cripto Institucional
- BTC-USD, ETH-USD
- ETFs: IBIT (BlackRock BTC), FBTC (Fidelity)
- MicroStrategy (proxy BTC): MSTR
- Coinbase (exchange): COIN
- Hashdex / 21Shares (ETPs europeos): web_search

Métricas: dominio BTC, TVL DeFi, flujos ETFs, hash rate.
